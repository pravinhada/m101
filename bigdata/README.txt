This folder contains 3 files for the Big Data Tutorial
1. data.tar.gz = contains files for the map reduce application
2. mincemeat.py = is the lightweight python application for map reduce
3. read_data.py  = is the map reduce for reading all the files in data and count the words by author

In order to run the application:
first run the python script as
>python file_name.py

Then start the client
>python mincemeat.py -p changeme localhost
